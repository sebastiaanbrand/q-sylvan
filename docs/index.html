Q-Sylvan

Quantum circuit simulation with parallel decision diagrams

What is Q-Sylvan?

Q-Sylvan extends the parallel decision diagram library Sylvan (v1.4.0) with QMDDs, as well as functionality to simulate quantum circuits. This is currently still a beta version.

Getting started

Installation instructions and examples can be found on the main GitHub page.
